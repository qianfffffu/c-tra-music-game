# 五音入弦

Repository: `c-tra-music-game`

一个非常轻量的中国传统音律网页音游原型。项目用纯 HTML、CSS、JavaScript 实现，不依赖构建工具或外部音频资源。

## 直接游玩

GitHub Pages 发布后，直接访问：

https://qianfffffu.github.io/c-tra-music-game/

如果页面暂时 404，请在仓库 `Settings` -> `Pages` 中把 `Deploy from a branch` 设置为 `main` / `/root`，等待 GitHub Pages 部署完成。

## 本地开发

双击 `index.html` 可预览。若浏览器限制本地音频上下文，可用任意静态服务启动后访问：

```bash
python3 -m http.server 4173
```

## 玩法

- 基础乐器：古琴、古筝、扬琴。
- 进阶乐器：竖琴、编钟。
- 五音键位：`A` 宫、`S` 商、`D` 角、`F` 徵、`G` 羽。
- 变奏音：下落音符带“变”标记时，按住 `Shift` 再按对应五音键。
- 曲目：根据传统古琴名曲热度和大众识别度整理为轻量游戏谱面。

## 曲库说明

初版优先收录《高山流水》《梅花三弄》《平沙落雁》《潇湘水云》《广陵散》《阳关三叠》等传统名曲。谱面是面向音游节奏的五声音阶动机整理，不是完整古琴演奏谱，也不复制商业音频。

参考来源：

- 排行榜123网：《古琴十大传统名曲》提到现存琴谱与传统名曲整理。
- 扬州久音古琴坊：《中国古琴十大名曲》列出潇湘水云、广陵散、高山流水、渔樵问答、平沙落雁、阳春白雪、胡笳十八拍、阳关三叠、梅花三弄、醉渔唱晚。
- 买购网：《古琴十大名曲》列出潇湘水云、高山流水、渔樵问答等大众熟悉曲目。

## 视觉方向

首屏是 2D-live 风格封面页：右侧为原创黑长直大师姐在高山云雾中抚琴，发丝、云雾和风带通过 CSS/SVG 动画实现；左侧完成乐器选择、乐曲选择和开始游戏。进入游戏后切换为暗色调弹奏界面，白色发光五音字符下落。
