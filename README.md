# 五音入弦

一个非常轻量的中国传统音律网页音游原型。项目用纯 HTML、CSS、JavaScript 实现，不依赖构建工具或外部音频资源。

## 运行方式

直接打开 `index.html` 即可运行。也可以用任意静态服务启动：

```bash
python3 -m http.server 4173
```

然后访问 `http://localhost:4173`。

## 玩法

- 基础乐器：古琴、古筝、扬琴。
- 进阶乐器：竖琴、编钟。
- 五音键位：`A` 宫、`S` 商、`D` 角、`F` 徵、`G` 羽。
- 变奏音：下落音符带“变”标记时，按住 `Shift` 再按对应五音键。
- 曲目：根据传统古琴名曲热度和大众识别度整理为轻量游戏谱面。

## 曲库说明

初版优先收录《高山流水》《梅花三弄》《平沙落雁》《潇湘水云》《广陵散》《阳关三叠》等传统名曲。谱面是面向音游节奏的五声音阶动机整理，不是完整古琴演奏谱，也不复制商业音频。

参考来源：

- 排行榜123网：《古琴十大传统名曲》提到现存琴谱与传统名曲整理。
- 扬州久音古琴坊：《中国古琴十大名曲》列出潇湘水云、广陵散、高山流水、渔樵问答、平沙落雁、阳春白雪、胡笳十八拍、阳关三叠、梅花三弄、醉渔唱晚。
- 买购网：《古琴十大名曲》列出潇湘水云、高山流水、渔樵问答等大众熟悉曲目。

## 视觉方向

整体采用简洁典雅的米纸、墨绿、朱印、鎏金配色。页面中的“黑长直大师姐”是原创国风引导师形象，用内联 SVG 绘制，不使用第三方角色图或商业素材。
